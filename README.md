# Expense Tracker App

This project is a full-stack expense tracking web application built to help users manage personal finances in a simple and modern way. It allows users to register, add expenses, view spending trends, and get AI-based financial suggestions.

## What this project does

- Lets users create an account and log in securely
- Helps users add, edit, and delete expenses
- Shows spending insights through charts and summary cards
- Includes an AI Advisor feature for simple money guidance
- Provides a clean and responsive user interface

## Tech Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: MongoDB with Mongoose
- Authentication: JWT + bcryptjs
- AI Integration: Gemini / Claude-based advisor support

## Main Features

- User authentication and onboarding
- Expense management with categories and notes
- Dashboard with income, expenses, and balance overview
- Visual charts for spending analysis
- AI-powered financial suggestions

## How to Run the Project

### 1. Backend

```bash
cd backend
npm install
npm run dev
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Make sure your backend is connected to MongoDB and the frontend points to the correct API URL.

## Project Structure

- backend: server, routes, authentication, database models
- frontend: pages, components, hooks, and UI design

## Why this project is valuable

This project is a strong example of a modern full-stack application. It combines user authentication, database operations, analytics, and AI features in one complete product that is easy to understand and demonstrate.
